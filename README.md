## О проекте

`CSS-progect` — учебный репозиторий с последовательными заданиями по основам CSS.  
Каждая работа посвящена отдельной теме и содержит HTML-разметку, таблицу стилей и, где это необходимо, дополнительные ресурсы: изображения или локальные шрифты.

Главная страница репозитория демонстрирует базовое влияние CSS на:

- цвет текста;
- начертание шрифта;
- размер текста.

Остальные задания находятся в отдельных ZIP-архивах.

## Изученные темы

| Задание | Тема | Основные техники |
|---|---|---|
| 1 | Введение в CSS | Цвет, жирность и размер текста |
| 2 | Селекторы | Селекторы тегов, классов, ID, атрибутов и комбинаторы |
| 3 | Блочная модель | `display`, размеры, `padding`, `border`, `margin`, `box-sizing` |
| 4 | Отступы | Внешние отступы, сокращённая запись и отрицательные значения |
| 5 | Нормализация стилей | Сброс браузерных стилей для текста, списков, кнопок и полей |
| 6 | Шрифты | `@font-face`, локальный Roboto, начертания и наследование |
| 7 | Цвет | HEX, RGB/RGBA, HSL, `transparent` и `currentColor` |
| 8 | Фон и градиент | Фоновое изображение, прозрачный фон и `linear-gradient()` |
| 9 | Рамки, тени и фильтры | `border`, `border-radius`, `outline`, `drop-shadow()` |
| 10 | Позиционирование | `relative`, `absolute`, `fixed` и `z-index` |

В репозитории также есть отдельный пример со стилизацией псевдоэлементов:

- `::placeholder`;
- `::selection`;
- `::file-selector-button`;
- `::before` и `::after`;
- `::marker`;
- `::first-letter`;
- `::first-line`.

## Примеры изученных CSS-конструкций

### Селекторы

```css
h1 {
  color: yellow;
}

.purple-text {
  color: blueviolet;
}

#orange-text {
  color: orange;
}

a[href*="scam"] {
  color: red;
}

.protected-links a[href^="https://"] {
  color: green;
}
```

### Блочная модель

```css
.box {
  width: 120px;
  height: 120px;
  padding: 10px;
  border: 10px solid black;
  box-sizing: border-box;
}
```

### Подключение локального шрифта

```css
@font-face {
  font-family: "Roboto";
  src: url("./fonts/Roboto-Regular.ttf") format("truetype");
  font-weight: 400;
  font-style: normal;
}
```

### Градиент

```css
button {
  background-image: linear-gradient(90deg, #e91e63, #00bcd4);
}
```

### Позиционирование

```css
.box {
  position: fixed;
  right: 0;
  bottom: 0;
}
```

## Как открыть проект

Для просмотра главного задания достаточно открыть файл:

```text
index.html
```

двойным нажатием в проводнике или через браузер.

Для работы с отдельным заданием:

1. Распакуйте нужный ZIP-архив.
2. Не меняйте взаимное расположение HTML, CSS, изображений и шрифтов.
3. Откройте содержащийся внутри файл `index.html`.

## Запуск через локальный сервер

Локальный сервер не обязателен, но удобен при работе в редакторе.

### Python

В терминале из папки с распакованным заданием выполните:

```bash
python -m http.server 8000
```

На некоторых системах используется команда:

```bash
python3 -m http.server 8000
```

После запуска откройте:

```text
http://localhost:8000
```

### Visual Studio Code

Можно использовать расширение **Live Server**:

1. Откройте папку задания в Visual Studio Code.
2. Нажмите правой кнопкой по `index.html`.
3. Выберите **Open with Live Server**.

## Цель проекта

Репозиторий создан для практического закрепления основ CSS:

- понимания каскада и наследования;
- выбора элементов разными типами селекторов;
- управления размерами и отступами;
- подключения шрифтов;
- оформления текста и фона;
- работы с рамками, тенями и фильтрами;
- позиционирования элементов на странице.
